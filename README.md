Ainglo 🚀

Ainglo es un asistente inteligente para aprender inglés mediante inteligencia artificial, enfocado especialmente en hispanohablantes que desean mejorar su speaking, pronunciación y vocabulario de una manera práctica, natural y accesible.

El proyecto combina modelos de IA conversacional, reconocimiento de voz y generación de audio para crear una experiencia de aprendizaje personalizada directamente desde Telegram.

✨ Características principales
📚 Diccionario inteligente con traducciones, ejemplos y contexto real.
🎙️ Corrección de speaking mediante análisis de audios.
🧠 Memoria personalizada para recordar errores y progreso.
🔊 Generación de pronunciación correcta usando TTS.
💬 Conversaciones naturales en inglés con IA.
🤖 Integración con Telegram para acceso rápido y sencillo.
🆓 Uso de herramientas open source y modelos locales para minimizar costos.
🛠️ Tecnologías utilizadas
Python
Telegram Bot API
python-telegram-bot
Faster Whisper
Ollama
Llama 3 / Mistral / Gemma
Piper TTS
SQLite (inicialmente)
🎯 Objetivo del proyecto

Construir una plataforma inteligente de aprendizaje de inglés que ayude a mejorar speaking y pronunciación de forma personalizada, utilizando inteligencia artificial local y herramientas gratuitas.

📌 Estado del proyecto

Actualmente en fase de planificación y desarrollo del MVP.
Las primeras funcionalidades incluirán:

traducción inteligente,
corrección de pronunciación,
conversaciones básicas,
y respuesta por voz.
🌎 Visión a futuro

Ainglo busca evolucionar hacia una plataforma completa de aprendizaje asistido por IA, incluyendo:

modos especializados,
análisis avanzado de speaking,
shadowing,
panel web,
aplicación móvil,
y sistema premium escalable.
